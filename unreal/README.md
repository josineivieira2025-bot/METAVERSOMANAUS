# Manaus Online Unreal Engine 5

Este diretorio contem a base correta para o jogo AAA em Unreal Engine 5.

Ele nao contem assets placeholder de baixa qualidade. A producao visual real deve importar:

- MetaHuman ou personagens humanos equivalentes.
- Veiculos AAA com Chaos Vehicles.
- Mapa 1:1 produzido por pipeline GIS/OSM.
- Predios modelados individualmente.
- Materiais PBR 4K.
- Animacoes Motion Matching/IK.

## Abrir

1. Instale Unreal Engine 5.4 ou superior.
2. Abra `ManausMetaverse.uproject`.
3. Gere os arquivos de projeto C++ se o Unreal pedir.
4. Compile pelo Unreal Editor ou Visual Studio.
